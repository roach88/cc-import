# cc-import

A [Hermes Agent](https://github.com/NousResearch/hermes-agent) plugin that imports [Claude Code](https://claude.com/claude-code) plugins (skills + agents) into a local Hermes installation. Translates Claude Code sub-agent personas into Hermes delegation skills, and copies skill bundles into Hermes's skills tree, with manifest-based idempotency that preserves user edits across re-imports.

> **Pre-release.** Slice 1 of three is in active development — see [`docs/plans/`](docs/plans/) for the breakdown. The full README lands with slice 1's final unit.
